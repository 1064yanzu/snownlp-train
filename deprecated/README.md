# Deprecated Files / 归档文件

此文件夹包含开发过程中的过渡版本和调试工具。这些文件的功能已被整合到主工具中，保留备用。

## 文件说明

| 文件名 | 说明 |
|--------|------|
| `train.py` | 早期训练脚本 |
| `train_fixed.py` | 修复版训练脚本v1 |
| `train_fixed_v2.py` | 修复版训练脚本v2 |
| `train_simple_fix.py` | 简单修复脚本 |
| `test_new_model.py` | 英文名测试脚本 |
| `修复版GUI工具.py` | GUI调试工具 |
| `测试进度界面.py` | UI测试工具 |
| `诊断模型保存问题.py` | 问题诊断工具 |
| `手动替换模型.py` | 手动模型替换工具 |
| `数据检查工具.py` | 数据标签分析工具 |
| `三分类示例.py` | 三分类使用示例 |

## 使用建议

- 日常使用请使用根目录的主工具
- 如遇特殊问题，可参考这些工具的实现
- `数据检查工具.py` 和 `三分类示例.py` 仍可独立使用
